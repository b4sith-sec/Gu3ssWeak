# Security Policy

## Scope of This Project

Gu3ssWeak is an intentionally vulnerable Android application for educational mobile security practice. All vulnerabilities are deliberate and documented.

## Reporting Issues With This Project

If you find a bug in the app itself, not one of the 19 documented vulnerabilities, or inaccurate documentation, please open a GitHub issue with reproduction steps.

## What NOT to Report

Do not report any of VULN-WV, VULN-DL, VULN-AP, VULN-BR, VULN-SV, or VULN-SQL-01 issues. These are intentional.

## Disclaimer on Real-World Use

Only test applications and systems you own, or for which you have explicit written authorization, such as a bug bounty scope or pentest engagement letter. Unauthorized access to computer systems is illegal in most jurisdictions including the Computer Fraud and Abuse Act and Computer Misuse Act. This project's authors are not responsible for misuse against unauthorized systems.

## Responsible Disclosure General Guidance

If you find a real vulnerability in a third-party app while practicing:

1. Check for a published security policy or bug bounty program.
2. Report privately to the vendor first.
3. Provide clear reproduction steps and impact assessment.
4. Allow a reasonable disclosure timeline, commonly 90 days, before public write-up.
